# FNO Wealth Planner

A local-first wealth planning app for tracking:

- monthly income
- recurring expenses and payments
- asset allocations across crypto, stocks, real estate, and cash
- long-term projected portfolio growth

## Run locally

Open `index.html` in your browser.

For a better local development workflow, you can also use the VS Code Live Server extension or any static file server.

## Features

- net worth dashboard
- monthly cash-flow tracking
- recurring payments overview
- asset planner with expected annual return and monthly contributions
- 1, 5, and custom-year projection summary
- built-in projection chart
- local persistence with `localStorage`

## Notes

- This is a planning tool, not financial advice.
- Projection math is based on fixed monthly contributions and fixed expected annual returns.
